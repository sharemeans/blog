## 重命名函数或者变量
```
F2 + 点击变量名（或函数名）：会出现一个输入框可修改变量名
```


## 复制并粘贴行
```
shift + Alt + 下箭头：复制并粘贴到下面一行
shift + Alt + 上箭头：复制并粘贴到上面一行
```

## 复制光标（多行同时修改）
```
ctrl + Alt + 上箭头：在上一行同一列生成一个光标
ctrl + Alt + 下箭头：在下一行同一列生成一个光标
```

## 标签展开折叠
```
ctrl + shift + [：折叠
ctrl + shift + ]：展开
```
